After one month at 42 School and to better prepare for my first exam, I developed this program to review fundamental concepts such as memory allocation/free, multiple structure management, and header manipulation.

## Getting Started

</div>

### Clone & Setup
```bash
# Clone repository
git clone git@github.com:Nesplee/CRYPTO.git
cd CRYPTO

# Install required packages
sudo apt-get update
sudo apt-get install gcc make python3-pip
pip3 install dash pandas plotly
```
### Build & Launch
```bash
# Compile the program
cd CRYPTO.V3
make

# Run the Program
Option 1: Command Line Mode
./asset_manager

Option 2: Window Mode
./launcher
```
